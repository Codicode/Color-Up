jQuery Image Color Up Version 1

Usage :<img src="my_image.jpg" class="colorup" effect="negatif" speed="1000" inverse="true"/>
Effect :mono, sepia, negative, light, dark. mono is the default value.
Speed :Animation/Transition duration, in milliseconds. 1000ms is the default value.
Inverse :Which image to start with. false: start with the original image and it is the default value.
